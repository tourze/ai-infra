# ai-infra skills

本仓库用于集中维护本地 `skills/`，并一键软链到常用编码代理目录（`cc`/`codex`）。

## 本地结构

- 技能目录：`./skills`
- 映射文件：`./skills-map.txt`
- 同步脚本：`./sync-skills.py`

## 常用命令

```bash
# 1) 按映射同步远程 skills 到本地 ./skills
python3 sync-skills.py

# 2) 一键把本仓库 skills 软链到 cc + codex
npm run link:skills
```
